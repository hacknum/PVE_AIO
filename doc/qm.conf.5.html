NAME 名字

qm.conf - Proxmox VE Virtual Machine Configuration qm.conf - Proxmox VE虚拟机配置

SYNOPSIS 概要

/etc/pve/qemu-server/.conf

DESCRIPTION 描述

The /etc/pve/qemu-server/.conf files stores VM configuration, where "VMID" is the numeric ID of the given VM. /etc/pve/qemu-server/.conf 文件存储 VM 配置，其中“VMID”是给定 VM 的数字 ID。

File Format 文件格式

The file uses a simple colon separated key/value format. Each line has the following format: 该文件使用简单的冒号分隔键/值格式。每行都有以下格式：

xxxxxxxxxx
OPTION: value

Blank lines in the file are ignored, and lines starting with a # character are treated as comments and are also ignored. 文件中的空行将被忽略，以 # 字符开头的行将被视为注释，并且也会被忽略。

One can use the qm command to generate and modify those files. 可以使用 qm 命令来生成和修改这些文件。
